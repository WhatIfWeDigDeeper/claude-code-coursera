Notes for screenshot capture:
- Use consistent browser window size
- Capture in both light and dark modes where applicable
- Ensure UI is clean (no personal data, realistic examples)
- Use high resolution (at least 1920x1080 for desktop views)
- Include browser chrome if showing full app context
- Crop tightly if showing specific components
- Use annotation tools to highlight specific elements (arrows, boxes)
- Save in PNG format for quality
- Consider animated GIFs for multi-step processes

Recommended tools:
- macOS: Cmd+Shift+4 for screenshots, Skitch for annotations
- Windows: Snipping Tool or Snip & Sketch
- Browser: Browser DevTools for responsive screenshots
- Annotation: Snagit, Greenshot, or built-in markup tools
